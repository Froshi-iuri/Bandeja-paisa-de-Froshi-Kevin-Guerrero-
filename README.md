# Bandeja-paisa-de-Froshi-Kevin-Guerrero-
sistema súper básico de restaurante.
este sistema es un sistema basico que simula un restaurante
no se usó while por lo que se finaliza al primer pedido.

el sistema lleva tuplas para que el nombre del restaurante y la calle no se cambien por error, 
diccionarios para guardar los platos con su número para que sea fácil encontrarlos y sets para las categorías como jugos o comidas 
y que no se repitan los nombres. también usa listas para guardar el historial de todo lo que el usuario va haciendo paso a paso, 
ciclos for para que el programa muestre solo el menú y el historial sin escribir tanto print, el input para poder escribir el pedido desde el teclado y f-strings 
para que los mensajes se vean bien ordenados con las variables.
para usarlo solo tienes que ejecutar el archivo, escribir tu pedido y al final el programa te muestra el resumen de lo que pasó.
